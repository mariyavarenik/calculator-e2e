## e2e selenium tests examples

### js
```bash
  cd js/
  npm install
  npm run test
```

### python
```bash
  cd python/
  python3 -m venv env
  source env/bin/activate
  pip install -r requirements.txt
  wget https://chromedriver.storage.googleapis.com/93.0.4577.15/chromedriver_mac64.zip 
  unzip chromedriver_mac64.zip
  python main.py
```
